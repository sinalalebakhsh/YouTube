        126- Building a Basic Theme
==================================================➤

A basic Hugo theme only needs these files:

layouts
├── _default
│   ├── list.html
│   └── single.html
└─── index.html

    # The index.html file is the home page of the site.
    # The _default directory contains the default layouts applied to single content pages (single.html)
    # pages that display lists of content pages (list.html)
==================================================➤
            
                Generating the Theme

Hugo has a built-in theme generator that creates everything you need for your theme.
==================================================➤
